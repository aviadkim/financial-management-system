# מערכת ניהול פיננסי - FinanceFlow

## סקירה כללית
מערכת לניהול שיחות ופגישות עם לקוחות פיננסיים, כולל הקלטה, תמלול ומעקב רגולטורי.

## סטטוס נוכחי
- ✅ Phase 1.1: ניהול לקוחות
  - טופס הוספת לקוח חדש
  - הצגת פרטי לקוח
  - ניהול נתוני לקוחות

- ✅ Phase 1.2: מערכת הקלטות
  - הקלטת שיחה בזמן אמת
  - העלאת הקלטות קיימות
  - תמלול בסיסי
  - מעקב אחר שאלות רגולטוריות
